# Direktori Aset iSURF

Direktori ini berisi berkas multimedia dan aset statis yang digunakan untuk keperluan dokumentasi, desain, maupun presentasi sistem iSurf Lab.

Berisi:
- Logo Institusi (IPB / iSurf)
- Tipografi / Font (*compiled*)
- Diagram beresolusi tinggi (*exported PNG/JPG*)
- Hasil tangkapan layar antarmuka pengguna (UI/UX Mockups)

File di sini umumnya **bukan** aset yang digunakan secara *runtime* oleh aplikasi web (aset web berada di `apps/web/frontend/web/`).
